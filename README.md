# IIIF INTERIM Annotator

Un annotatore semantico basato su IIIF per la gestione e l'analisi di fenomeni intermediali, con particolare focus sull'ecfrasi.

## Descrizione

Questo progetto fornisce strumenti per l'annotazione semantica di relazioni intermediali utilizzando il framework IIIF (International Image Interoperability Framework) e l'ontologia INTERIM.

## Ontologie Utilizzate

- **INTERIM**: Ontologia core per relazioni intermediali
- **MIRO**: Media Interactions and Relations Ontology
- **GEKO**: Ontologia per la modellazione dell'ecfrasi
- **MLAO**: Multi-Level Annotation Ontology
- **ICON**: Iconography Ontology
- **HICO**: Historical Context Ontology
- **CIDOC-CRM**: Conceptual Reference Model
- **LRMoo**: FRBR Object-Oriented

## Contenuto della Repository

- `interim.ttl`: Ontologia INTERIM completa in formato Turtle
- `*.json`: Annotation lists IIIF per storyboard
- `model interim.jpg`: Modello concettuale di riferimento
